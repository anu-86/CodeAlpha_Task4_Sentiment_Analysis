# CodeAlpha Task 4 - Sentiment Analysis

## Project Overview

This project performs **Sentiment Analysis** on Amazon customer reviews using **Natural Language Processing (NLP)**. The objective is to classify customer reviews into **Positive**, **Negative**, or **Neutral** sentiments using the **VADER Sentiment Analyzer** from the NLTK library.

The project also visualizes the sentiment distribution using bar and pie charts to better understand customer opinions.

---

## Features

- Load and analyze Amazon review data
- Perform sentiment analysis using VADER (NLTK)
- Classify reviews as Positive, Negative, or Neutral
- Visualize sentiment distribution with charts
- Understand customer opinion through review analysis

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- NLTK (VADER Sentiment Analyzer)
- Jupyter Notebook

---
